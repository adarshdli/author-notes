## Install
``` python 
python -m pip install --upgrade pip
pip install pymdown-extensions
pip install mkdocs
pip install mkdocs-awesome-pages-plugin
pip install mkdocs-material==5.0.1
```

## Serve command
``` python 
mkdocs serve
```

## Build command
``` python 
mkdocs build --clean
```
## Publish to Github
``` python 
mkdocs gh-deploy --force
```
